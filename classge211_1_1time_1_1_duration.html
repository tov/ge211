Duration Class Reference

Detailed Description

A length of time.

This is an opaque type representing a high-precision segment of time that can be extracted as a double of seconds by the seconds() const member function. A Duration can be constructed from the number of seconds, also as a double. Duration values can also be compared, added, subtracted, and added and subtracted to Time_point values.
