◆ make_rectangle()

Basic_rectangle<T> ge211::geometry::make_rectangle ( T  x,
T  y,
T  width,
T  height 
)

Constructs a Basic_rectangle given its member variables.

It takes them in the same order as the constructor: the x and y coordinates followed by the width and height.

Unlike the constructor, this function can infer the coordinate type from its arguments. For example:

// infers Basic_rectangle<double>:
auto unit_square = make_rectangle(0., 0., 1., 1.);
// infers Basic_rectangle<int>:
auto big_reversi = make_rectangle(0, 0, 16, 16);
